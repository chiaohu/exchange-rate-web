# 匯率換算網站
> 此專案作品為串接API製作簡易的匯率轉換網站。

## DEMO

[exchange](https://chiaohu.github.io/exchange-rate-web/dist/#/)

### 簡介
由於此專案有提到希望應用computed功能與串接API的功能，就我的理解computed比較傾向於所選取的資料有變動時候立即做計算，用匯率數字也可以直接連結API取得因此我認為計算匯率應用應該蠻適合的。

## 此專案運用了
* Vue cli webpack
* 載入bootstrap與scss樣式
* 將Navbar製作成component
* 應用computed功能即時計算匯率轉換
* 可切換至不同匯率頁面
* 串接買入與賣出的api資料
